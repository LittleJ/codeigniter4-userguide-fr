Bienvenue sur CodeIgniter4¶

CodeIgniter est un Framework pour le développement d’applications - une boîte à outils - pour les gens qui créent des sites Web en utilisant PHP. Son objectif est de vous permettre de développer des projets beaucoup plus rapidement que si vous écriviez du code à partir de zéro, en fournissant un riche ensemble de bibliothèques pour les tâches courantes, ainsi qu’une interface simple et une structure logique pour accéder à ces bibliothèques. CodeIgniter vous permet de vous concentrer de manière créative sur votre projet, en minimisant la quantité de code nécessaire pour une tâche donnée.

Dans la mesure du possible, CodeIgniter est resté aussi flexible que possible, vous permettant de travailler comme vous le souhaitez, sans être forcé de travailler d’une manière pré-déterminée. Le coeur de ce Framework peut être facilement étendu, ou complètement remplacé, pour que le système fonctionne de la manière dont vous le souhaitez. En bref, CodeIgniter est un Framework flexible, qui essaie de fournir les outils dont vous avez besoin, tout en ne bloquant votre avancement.

A qui s’adresse CodeIgniter?¶

CodeIgniter vous conviendra si:

  • Vous souhaitez un Framework avec une faible emprunte.

  • Vous avez besoin de performances exceptionnelles.

  • Vous souhaitez un Framework qui requière presque aucune configuration.

  • Vous souhaitez un Framework qui ne requière par l’utilisation de lignes de commande.

  • Vous souhaitez un Framework qui ne requièrent pas d’adhérer à des règles de développement strictes.

  • Vous n’êtes pas intéressé par les bibliothèques monolithiques à large échelle tel que PEAR.

  • Vous ne voulez pas être forcé d’apprendre un language de templating (bien qu’un parser de modèle soit disponible optionnellement si malgré tout vous le souhaitez).

  • Vous évitez la complexité et privilégiez des solutions simples.

  • Vous avez besoin d’une documentation claire et complète.
